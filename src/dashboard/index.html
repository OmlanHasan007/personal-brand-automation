Shadowing

Beginner → advanced track
Level 1 — Foundation
• Learn 10 ML terms and define each in plain English
• Record yourself explaining "what is machine learning" (2 min)
• Read one HF blog post aloud every day
Level 2 — Conversations
• Explain a project to an imaginary non-technical recruiter
• Answer 5 behavioural questions in English using STAR
• Join Hugging Face / LangChain Discord — write 3 comments
Level 3 — Interview-ready
• Full 30-min mock interview in English (use interview coach)
• Record a 5-min project demo video — post on LinkedIn
• Reduce "um/uh" filler words to <3 per answer
