STAR template
Situation → Task → Action → Result
Target: 90–120 seconds per answer
Key phrases
Buying time:
"That's a great question — let me think through it."

Showing process:
"My first instinct is X, but the trade-off is..."

Recovering:
"I haven't used that specific tool, but my approach would be similar to how I handled..."

Closing answer:
"Does that answer it, or would you like more depth on any part?"
Daily practice plan (20 min/day)
Monday
Shadowing
Pick a 3-min AI YouTube video. Play sentence by sentence — repeat out loud matching rhythm. Try: Andrej Karpathy, Lex Fridman, 3Blue1Brown.
Wednesday
Explain a concept
Set a 3-min timer. Explain any ML topic out loud in English to a non-expert. Record on phone — spot your filler words.
Friday
Mock answer
Pull a random interview question from the coach tab. Answer out loud with STAR structure. Time it — aim for 90–120 seconds.
